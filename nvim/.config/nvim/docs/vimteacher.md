# VimTeacher 插件说明

## 插件信息

- 名称：`brentsec/VimTeacher`
- 本地入口文件：`lua/plugins/vimteacher.lua`
- 定位：交互式 Vim 教学/练习插件

## 安装

在 Neovim 中执行：

```vim
:Lazy sync
```

安装完成后可用以下方式启动。

## 用法

- 命令：`:VimTeacher`
- 快捷键：`<leader>tv`

进入后按界面提示完成练习任务即可。

## 建议训练节奏

1. 每天 10-15 分钟，专练移动与文本对象。
2. 先追求正确率，再追求速度。
3. 把练习中常用动作迁移到真实代码编辑中。
