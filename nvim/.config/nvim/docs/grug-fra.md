# 🔹 grug-far.nvim 常用操作说明

| 功能               | 快捷键                 | 说明                                |
| ---------------- | ------------------- | --------------------------------- |
| 打开帮助             | `g?`                | 弹出 grug-far 内部帮助窗口，显示快捷键和操作说明     |
| 执行替换             | `\r`                | 对当前搜索结果执行替换（支持多行替换）               |
| 同步所有修改           | `\s`                | 将结果缓冲区里手动修改过的内容同步回对应文件            |
| 同步当前行修改          | `\l`                | 只同步当前结果行到文件                       |
| 同步当前行并跳到下一个      | `\n`                | 应用当前行修改，然后跳到下一条匹配                 |
| 同步当前行并跳到上一个      | `\p`                | 应用当前行修改，然后跳到上一条匹配                 |
| 同步当前文件           | `\v`                | 将缓冲区里当前文件的修改同步回原文件                |
| 打开历史窗口           | `\t`                | 显示历史搜索/替换列表，可以选择并再次执行             |
| 添加历史记录           | `\a`                | 将当前搜索/替换加入历史列表                    |
| 刷新搜索             | `\f`                | 重新触发搜索（当外部文件被修改时非常有用）             |
| 跳转到匹配位置          | `Enter`             | 光标在路径上 → 打开文件；光标在匹配行 → 跳转到文件对应行/列 |
| 打开匹配但留在 grug-far | `\o`                | 和 `Enter` 类似，但不会离开 grug-far 缓冲区   |
| 上/下一个匹配          | `↑` / `↓`           | 移动光标到上/下一个搜索结果，并触发 Open           |
| 应用当前匹配并跳下一个      | `\j`                | 应用当前行修改，然后移除并跳到下一个匹配              |
| 应用当前匹配并跳上一个      | `\k`                | 应用当前行修改，然后移除并跳到上一个匹配              |
| 发送到 quickfix     | `\q`                | 将匹配行发送到 quickfix 列表，删除行后不会被包含     |
| 中止操作             | `\b`                | 取消当前操作（例如搜索太大或者替换中途想退出）           |
| 关闭 grug-far 窗口   | `\c`                | 关闭 grug-far 缓冲区，如果有替换/同步在进行，会提示确认 |
| 切换搜索引擎           | `\e`                | 和下一个搜索引擎切换（如果配置了多个）               |
| 显示/隐藏搜索命令        | `\w`                | 切换是否显示实际执行的搜索命令（调试用）              |
| 预览匹配位置           | `\i`                | 在浮动窗口预览匹配文件的上下文                   |
| 切换替换解释器          | `\x`                | 切换替换解释器，例如用 lua 脚本生成每条匹配的替换内容     |
| 输入框切换            | `Tab` / `Shift+Tab` | 在搜索/替换输入框间切换，循环使用                 |

# 🔹 使用小技巧

1. **快速替换整个项目**

   ```
   <leader>sr → 输入搜索 → Tab 切换到替换 → 输入替换 → \r 或 \s 执行
   ```

2. **只修改某几行**

   * 手动删除缓冲区不需要修改的结果行
   * 然后按 `\s` 或 `\v` 同步

3. **快速跳转浏览结果**

   * 光标在结果行按 `Enter` 或 `\o` → 查看匹配上下文
   * 光标上下移动 `↑ ↓` 浏览匹配

4. **多行替换或脚本生成替换**

   * 用 `\x` 切换替换解释器（Lua 等）
   * 可以对每条匹配生成不同内容

5. **历史管理**

   * `\t` 打开历史 → 选择老搜索/替换快速复用
   * `\a` 保存当前操作到历史
